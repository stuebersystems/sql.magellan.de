# View **ClassCourses**

## Query

??? info "SQL"

    ``` sql
    
      SELECT 
        CC."Mandant", CC."ID", CC."PrevID", CC."ClassTerm", KZ."Klasse", C."Term", CC."Course", 
        C."EnbreaID", C."Subject", C."SubjectStatus", C."SubjectType", 
        C."SubjectTeacher", C."Focus", C."Level", C."CourseNo", C."Bilingual", 
        C."Attribute"
      FROM "tblCourses" C
      LEFT JOIN "tblClassCourses" CC
      ON  CC."Mandant" = C."Mandant"
      AND CC."Course" = C."ID"
      LEFT JOIN "KlassenZeitraeume" KZ
      ON  KZ."Mandant" = CC."Mandant"
      AND KZ."ID" = CC."ClassTerm"
    ```
