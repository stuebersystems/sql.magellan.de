# View **LehrerSollBerechnung**

## Query

??? info "SQL"

    ``` sql
    
      SELECT T.*
      FROM "tblLehrerSollBerechnung" T  
      WHERE
        EXISTS(
          SELECT B.ID 
          FROM "tblBenutzer" B 
          WHERE 
            B."Mandant" = T."Mandant" AND 
            B."Kennung" = USER AND
            ((B."Lehrer" = T."Lehrer") OR
             (B."Gruppe1" = 0
               OR B."Gruppe1" = 1
               OR B."Gruppe1" = 4
               OR B."Gruppe1" = 5
               OR B."Gruppe1" = 9))
    )
    ```
