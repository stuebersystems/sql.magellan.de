# Ansicht **Lehrer**

## Abfrage

??? info "SQL"

    ``` sql
    
    SELECT T.*
    FROM "tblLehrer" T
    LEFT JOIN "tblBenutzer" B
    ON B."Mandant" = T."Mandant"
    WHERE
      B."Kennung" = USER AND
      ((B."Lehrer" = T.ID) OR 
       (B."Gruppe1" = 0 OR B."Gruppe1" = 1 OR
        B."Gruppe1" = 4 OR B."Gruppe1" = 5 OR
        B."Gruppe1" = 9 OR B."Gruppe1" = 10))
    ```
